# Approved Blog Tags

**Generated on:** 2026-01-12

## Purpose

This document defines the approved set of tags for blog posts. Each post should use **3 tags** to create meaningful topical connections and aid reader discovery of related content.

## Tag Selection Guidelines

- Choose 3 tags per post that best represent the core themes
- Tags should help readers discover related content
- Use exact tag names as listed below (case-sensitive)
- Prioritize tags that connect to existing content clusters
